### Problem
- Even Or Odd
- Take an integer value as input.
- Print "Even" if the number is even else print "Odd" if the number is odd.

### Note
- 0 will be considered as even number.

### Input Format
- Input single line.
- An integer value.

### Constraints
- -(10^5) <=N<= 10^5

### Output Format
- Output single line.
- A String "Even" Or "Odd".

```
Sample Input 0

98
Sample Output 0

Even
Sample Input 1

57
Sample Output 1

Odd
```
