### Problem
- Take integer N as input and print the following pattern.

### Input Format
- Single Line Input.
- Integer value

### Constraints
- 1<=N<=100

### Output Format
- N Lines of Pattern as shown in problem statement.

```
Sample Input 0

5
Sample Output 0

* * * * *
* * * * *
* * * * *
* * * * *
* * * * *
```
