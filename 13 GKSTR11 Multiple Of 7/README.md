### Problem
- Take an integer N as input, and print all the multiples of 7 till N(inclusive).

### Input Format
- Single line Input
- An integer value

### Constraints
- 0<=N<=10000

### Output Format
- Single line Output
- Integer values (All multiples of 7)

```
Sample Input 0

98
Sample Output 0

0 7 14 21 28 35 42 49 56 63 70 77 84 91 98
```
