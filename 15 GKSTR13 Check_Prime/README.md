### Problem
- Take integer N as input, and check whether N is prime or Not.
- Print "Prime" if the number is prime else print "Not a Prime".

### Note
- 1 is not a prime number.

### Input Format
- Single line Input
- An integer value

### Constraints
- 1<=N<=10000

### Output Format
- Single Line Output
- A string Showing "Prime" or "Not a Prime".

```
Sample Input 0

29
Sample Output 0

Prime
Sample Input 1

18
Sample Output 1

Not a Prime
```
