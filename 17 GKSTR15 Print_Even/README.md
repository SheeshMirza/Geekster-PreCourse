### Problem
- Given a integer n, print all even numbers from 0 till n (including, if even)

### Input Format
- Single line of Input
- An integer n

### Constraints
- 0 <= n <= 1000

### Output Format
- A range/series of even numbers from 0 till n, with each number in one line

```
Sample Input 0

10
Sample Output 0

0
2
4
6
8
10
```
